About This App

This application allows you to:

  • Upload your dataset (CSV, Excel, or TSV formats)
  • Preprocess your data with operations like handling missing values, scaling, and encoding
  • Explore your data with automatic exploratory data analysis (EDA)
  • Create various visualizations by selecting variables and plot types

Follow these steps to get started:

  1. Upload your dataset using the form above
  2. Review the data and apply preprocessing steps if needed
  3. Create visualizations by selecting the variables and plot type
  4. Download your processed data or visualizations as needed
